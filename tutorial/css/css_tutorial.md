# css tutorial

## 構文

タグごとに設定
```css
タグ名 {
	プロパティ: 値;
	...
}
```
```css
h1 {
	color: #123456;
	font-size: 20px;
}
```

クラス内(htmlで設定する)のタグだけに設定
```css
.クラス名 タグ名 {
	プロパティ: 値;
	...
}
```
```css
.footer li {
	color: #fff;
}
```

## ブロック要素、インライン要素

- ブロック要素
	- 親要素の幅と同じだけ広がる
	- 改行する
- インライン要素
	- 改行しない
	- 配置に影響を与えない(?)
