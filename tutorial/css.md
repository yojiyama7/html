### css

[様々なプロパティ](https://www.tagindex.com/stylesheet/properties/)

必要なものを調べて使う

#### 構文

タグごとに設定
```css
タグ名 {
	プロパティ: 値;
	...
}
```
```css
h1 {
	color: #123456;
	font-size: 20px;
}
```

クラス内(htmlで設定する)のタグだけに設定
```css
.クラス名 タグ名 {
	プロパティ: 値;
	...
}
```
```css
.footer li {
	color: #fff;
}
```

#### 文字

- 文字色;
	- `color: #257738;`

- 背景色
	- `background-color: #257738;`
	- `background-color: #fff;`
